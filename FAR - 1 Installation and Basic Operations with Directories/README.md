Shortcuts:
- cursors and [ENTER] to change directories
- [TAB] - change focus between panels
- [ALT+F1] - change drive (mounting point) on the left panel
- [ALT+F2] - change drive (mounting point) on the right panel
- [F7] - create directory
- [F5] - copy
- [F6] - move
- [F10] - finish workflow (quit)
